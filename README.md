# Coiffeur-Frontend
This project is a multi‑page frontend website designed with a feminine, elegant style tailored for ladies’ services. It includes:
- Home, Services, About, and Contact pages with a responsive navigation bar.
- Modern pastel theme with soft colors, rounded buttons, and clean typography.
- Interactive elements such as booking buttons, contact forms, and simple JavaScript alerts.
The goal is to provide a lightweight, user‑friendly interface that highlights services in a welcoming and stylish way.
